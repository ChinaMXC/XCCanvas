# XCCanvas

基于node.JS(Express) + Socket.IO的同屏展示程序
作者：[木小草](http:www.muxiaocao.cn)

> 互联网时代的同屏实时分享PPT，以跨地域、实时性、多互动为特点。 

## 使用场景：
	
	开会、网上授课等需要实时沟通的场景。

## 使用技术：

  node.js(Express框架) + Socket.io。

  例子：
